# Онлайн-курс по программированию — лендинг

Готовый одностраничный лендинг на Bootstrap 5 (CDN) с якорной навигацией и адаптивной версткой.

## Структура проекта

- `index.html` — основная страница лендинга
- `css/style.css` — минимальные пользовательские стили
- `assets/` — папка для изображений и медиа

## Локальный запуск

1. Скачайте или клонируйте репозиторий.
2. Откройте папку проекта.
3. Запустите `index.html` в браузере двойным кликом.
4. Для автоперезагрузки при разработке можно использовать расширение Live Server в VS Code.

## Публикация через GitHub Pages

1. Создайте репозиторий на GitHub и загрузите файлы проекта в ветку `main`.
2. Перейдите в `Settings` → `Pages`.
3. В блоке `Build and deployment` выберите `Deploy from a branch`.
4. Выберите ветку `main` и папку `/ (root)`, затем нажмите `Save`.
5. Дождитесь публикации и откройте выданный GitHub Pages URL.
