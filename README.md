# Онлайн-курс по программированию — лендинг

Готовый одностраничный лендинг на Bootstrap 5 (CDN) с якорной навигацией и адаптивной версткой.

## Структура проекта

- `index.html` — основная страница лендинга
- `css/style.css` — минимальные пользовательские стили
- `assets/` — папка для изображений и медиа
